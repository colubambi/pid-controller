# PID Controller Simulation

This project simulates a first-order system with P, PI, and PID controllers in MATLAB.
The code was created with guidance from ChatGPT.

## Features
- Simulate step response of a system with P, PI, and PID controllers
- Compare overshoot, rise time, and settling time
- Plot responses and save figures in `plots/` folder

## Tools Used
- MATLAB

## How to Run
1. Open `pid_simulation.m` in MATLAB
2. Run the script
3. Check the `plots/` folder for the saved plot
